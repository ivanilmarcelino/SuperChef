# SuperChef

## Sistema direcionado para o Ramo de Pizzaria 
Este projeto foi desenvolvido pelo Sr. Marcelo Neves, disponibilizado na pasta samples\application\Super no pacote da Minigui extend.

O objetivo deste trabalho será migrar o sistema de DBF para SQL, sem afetar suas funcionalidades mas poderá haver mudanças radicais nas telas, pois em sua maioria serão redesenhadas pelo Designer;
Todo o processo poderá ser acompanhado pelo Changelog, onde tentarei ser fiel em todas as mudanças, no entanto caso tente fazer e não der certo, entre em contato comigo, terei prazer em ajudar; [e-mail:](ivanil.marcelino@yahoo.com.br)
